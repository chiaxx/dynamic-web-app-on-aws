# 3. Create an HTTPS Listener
EC2 > Load Balancers > Select ALB[dev-ALB] > add listener
- protocol: HTTPS  | port:443
- Default Actions: Forward
   - target group: dev-TG
- Default SSL certificate > From ACM > select domain name[chia.click] > ADD
